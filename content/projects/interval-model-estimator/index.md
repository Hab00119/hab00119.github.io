A Python implementation of interval-based feature selection for spectroscopic data, designed to identify the most informative spectral regions for chemometric modeling. The method partitions spectra into intervals and evaluates each region's predictive contribution, reducing dimensionality while preserving signal relevant to the analyte of interest.

Developed to support milk protein quantification research, the tool integrates with standard scikit-learn pipelines and supports cross-validation-based interval scoring. It serves as a building block for automated preprocessing workflows in mid-infrared spectroscopy applications.
